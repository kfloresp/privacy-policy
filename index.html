Política de Privacidad

Última actualización: Abril 2026

Esta aplicación actúa como una bóveda segura diseñada para almacenar imágenes y documentos personales protegidos mediante autenticación biométrica.

1. Información que recopilamos

No recopilamos, almacenamos ni compartimos datos personales del usuario en servidores externos.

2. Uso de permisos

La aplicación utiliza los siguientes permisos:

  • Cámara (android.permission.CAMERA): Se utiliza únicamente para capturar imágenes que el usuario decide guardar dentro de la aplicación.
  • Almacenamiento: Para guardar imágenes y documentos localmente en el dispositivo.
  • Autenticación biométrica: Para proteger el acceso a la aplicación mediante huella digital u otros métodos seguros del dispositivo.

3. Uso de la información

Toda la información (imágenes y documentos) se almacena exclusivamente en el dispositivo del usuario. La aplicación no transmite datos a servidores externos.

4. Compartición de datos

No compartimos información con terceros.

5. Seguridad

La aplicación implementa mecanismos de seguridad como autenticación biométrica para proteger el acceso a los datos almacenados.

6. Eliminación de datos

El usuario puede eliminar sus datos en cualquier momento desinstalando la aplicación o eliminando manualmente los archivos desde la misma.

7. Cambios en esta política

Podemos actualizar esta política de privacidad ocasionalmente. Se recomienda revisar esta página periódicamente.
